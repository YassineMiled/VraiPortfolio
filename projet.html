FreeJobs


Ce projet a été réalisé du 2 décembre 2024 au 26 janvier 2025.

FreeJobs est un site web permettant aux utilisateurs de trouver des emplois gratuits en ligne. Le site offre la possibilité de filtrer les offres selon différents critères, tels que le niveau de compétence requis, le lieu de travail, le salaire, et bien plus encore.

Pour des raisons de confidentialité et de sécurité, je ne peux pas partager le code source de ce projet.

GSB Rapports de médecin


Le projet GSB vise à développer une application web destinée à faciliter la gestion des visites effectuées par les visiteurs médicaux auprès des professionnels de santé. Cette application permet de gérer les informations des médecins, d’organiser et de suivre les rapports de visite, et de présenter les médicaments du laboratoire. Les visiteurs médicaux peuvent y enregistrer les détails des visites, les produits proposés, les échantillons offerts, ainsi que les retours des praticiens. L’objectif principal est d’optimiser le suivi des interactions entre le laboratoire et les professionnels de santé tout en renforçant la promotion des produits pharmaceutiques. L’application repose sur une architecture moderne, utilisant Angular pour le front-end et PHP/MySQL pour le back-end.

Le projet est toujours en cours de développement. Vous trouverez le lien vers le GitHub du projet ci-dessous.
